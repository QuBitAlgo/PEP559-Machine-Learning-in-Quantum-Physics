# PEP559-Machine-Learning-in-Quantum-Physics

**last updated:** Jan 26, 2026

This is the official repository for the course **PEP 559: Machine Learning in Quantum Physics**. It contains lecture notes, slides, code demos, projects, and additional resources for learning and applying machine learning methods to problems in quantum physics.

## Course overview

This course provides a graduate-level introduction to core machine learning ideas and how they are used in modern quantum science. Topics include
- Fundamentals of machine learning and artificial intelligence
- Supervised and unsupervised learning
- Reinforcement learning
- Neural networks and deep learning
- Energy-based models
- Selected applications in quantum physics such as quantum error correction, quantum state classification, pattern storage and retrieval, quantum state representation and tomography, etc.
